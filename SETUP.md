# Furzi.com Setup Guide

## Required Assets

To complete the setup of your Furzi.com website, you'll need to add the following image assets to the `images/` directory:

### Hero Section Image
- **File**: `images/hero.jpg`
- **Purpose**: Background image for the hero section at the top of the page
- **Recommended size**: 1920x600px or larger
- **Suggested content**: Elegant clothing display or lifestyle image

### Product Images
Add the following product images to showcase your featured collection:

1. **File**: `images/product1.jpg`
   - Product: Classic Cotton Shirt
   - Recommended size: 600x600px

2. **File**: `images/product2.jpg`
   - Product: Premium Solid Shirt
   - Recommended size: 600x600px

3. **File**: `images/product3.jpg`
   - Product: Everyday Casual Wear
   - Recommended size: 600x600px

4. **File**: `images/product4.jpg`
   - Product: Luxury Linen Shirt
   - Recommended size: 600x600px

## Amazon Affiliate Links

Replace all instances of `YOUR_AMAZON_AFFILIATE_LINK` in `index.html` with your actual Amazon affiliate links:

1. Classic Cotton Shirt - Line ~196
2. Premium Solid Shirt - Line ~202
3. Everyday Casual Wear - Line ~208
4. Luxury Linen Shirt - Line ~214

## Quick Start

1. Add all required images to the `images/` directory
2. Update the Amazon affiliate links with your actual product URLs
3. Open `index.html` in a web browser to preview your site
4. Deploy to your hosting service (GitHub Pages, Netlify, etc.)

## Customization Tips

- Update color scheme by changing the golden accent color `#f5c400` throughout the CSS
- Modify product descriptions to match your actual affiliate products
- Add more products by copying the `product-card` div structure
- Update social media links in the footer
